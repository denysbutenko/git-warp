# Git-Warp v0.2.0

First public Git-Warp release.

Git-Warp is a Rust CLI for fast, safety-focused Git worktree management. This
release includes the daily worktree flow, terminal handoff, cleanup safeguards,
local agent-session visibility, and setup diagnostics.

## Highlights

- Create or switch worktrees with `warp switch <branch>` or `warp <branch>`.
- Run bare `warp` to open the interactive worktree switcher.
- Hand off to Terminal, iTerm2, Warp, the current shell, or echo/inplace modes.
- Inspect worktrees with primary, current, dirty, detached, and busy state.
- Preview and run cleanup with protected branches, process checks, interactive
  selection, and force/kill controls.
- Check setup with `warp doctor`.
- Install Claude/Codex hooks and open `warp agents` for local session visibility.
- Generate Bash, Zsh, and Fish completions.

## Install Or Upgrade

Fast path with Rust/Cargo already installed:

```bash
cargo install --locked --force --git https://github.com/denysbutenko/git-warp --tag v0.2.0 --bin warp git-warp
warp --version
warp doctor
```

Installer script:

```bash
curl -fsSL https://raw.githubusercontent.com/denysbutenko/git-warp/main/install.sh | sh
```

Source checkout:

```bash
git clone https://github.com/denysbutenko/git-warp
cd git-warp
git checkout v0.2.0
cargo install --path .
warp --version
warp doctor
```

Expected version output:

```text
warp 0.2.0
```

## Useful Commands

```bash
warp switch feature/my-change
warp ls
warp --dry-run cleanup --mode merged
warp cleanup --interactive
warp hooks-install --level user --runtime all
warp hooks-status
warp agents
```

## Notes

- Copy-on-Write acceleration is best on macOS/APFS. Other filesystems fall back
  to normal Git worktree creation.
- The installer script defaults to `v0.2.0`. Set `GIT_WARP_VERSION` before
  running it to install a different tag.
- Git-Warp is still a `0.x` tool. Use `warp doctor`, `--dry-run`, and focused
  command help when setting it up on a new repo.
